# OML 支持函数



## 内建函数

### 时间类



#### Time::now

获取当前日期时间

* 返回类型:  Meta::Time 

#### Time::now_date

获得当前日期

* 返回类型: Meta::digit

#### Time::now_hour

获得当前小时

* 返回类型: Meta::digit
